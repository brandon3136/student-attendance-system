# Overview
This is a simple web-based student attendance system designed to help teachers or administrators manage and track student attendance online. The system allows users to add students, mark attendance, and view attendance records through a user-friendly web interface.

## Features
- **Add Students**: Add new students to the system with details like name, ID, and class.
- **Mark Attendance**: Mark students as present or absent for a specific date.
- **View Attendance**: View attendance records for all students or filter by date.
- **User Authentication**: Secure login for teachers/administrators.
- **Responsive Design**: Works on desktop and mobile devices.
